# Random Quote Generator

A simple Flutter application that displays inspirational quotes randomly.

## Features

- Displays a random quote when the app starts
- Generate a new random quote with one tap
- Shows both quote and author
- Clean and minimal Material 3 UI
- Responsive layout
- Local quote data (no internet required)

## Tech Stack

- Flutter
- Dart
- Material 3

## Getting Started

```bash
git clone https://github.com/qazihasnat1122/random-quote-generator.git
cd random-quote-generator
flutter pub get
